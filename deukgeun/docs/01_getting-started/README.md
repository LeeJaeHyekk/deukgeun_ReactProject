## 🎯 빠른 시작 체크리스트

1. ✅ 프로젝트 클론
2. ✅ 의존성 설치 (`npm install`)
3. ✅ 환경 변수 설정 (`.env` 파일 생성)
4. ✅ 데이터베이스 설정 (MySQL 설치 및 DB 생성)
5. ✅ 개발 서버 실행 (`npm run dev`)
