## 🔧 주요 기능

### 공공 API 필터링
- 서울시 공공데이터 API
- 영업중인 헬스장만 필터링
- gym, gx, pt, 크로스핏 등 키워드 필터링

### 웹 크롤링
- 네이버, 구글, 다음 검색
- name 기반 크롤링
- 데이터 수집 및 병합

### 데이터 병합
- 공공 API + 웹 크롤링 데이터 통합
- 중복 제거
- 데이터 검증

### 히스토리 추적
- 크롤링 세션 관리
- 통계 생성
- 파일 저장
